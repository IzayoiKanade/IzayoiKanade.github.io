Map/Multimap

接下来说说map容器

·容器内所有元素都是pair(就是上面那个成对的)

·在pair中第一个元素为key（键值，就是个地址）；第二个元素为value（实值，就是存东西的）

·并且所有元素会根据key去排列，map/multimap也是个关联式容器，底层也是二叉树

·优点是很容易通过key值找到value值

·map和multimap的区别是，map可以放重复key值得元素，multi不行
